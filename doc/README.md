# Tile Explorer

> 三消解谜 · 打开即玩 · 无广告 · 免安装

## 这是什么？

Tile Explorer 是一款在浏览器中运行的三消瓦片解谜游戏。不需要下载 App，不需要注册账号——打开网页就能玩。

**核心玩法**：版面上堆叠着带有图案的瓦片。点击可用瓦片将其送入底部 7 格收集槽，凑齐 3 个相同图案自动消除。清空版面上所有瓦片即通关。

## 怎么玩？

### 基本操作

1. **点击瓦片** — 只有未被其他瓦片覆盖的才能点击（被覆盖的显示为半透明）
2. **自动消除** — 收集槽中出现 3 个相同图案时自动消除
3. **通关条件** — 清空版面所有瓦片
4. **失败条件** — 收集槽填满 7 格且无法匹配

### 道具

| 道具 | 图标 | 效果 | 解锁 |
|------|------|------|------|
| 洗牌 | 🔄 | 随机重排版面上所有瓦片的位置 | 第 1 关 |
| 撤销 | ↩️ | 把最后放入槽位的瓦片放回版面原位 | 第 1 关 |
| 提示 | 💡 | 高亮显示一组可以消除的瓦片（不影响三星评价） | 第 1 关 |
| 扔垃圾 | 🗑️ | 清空整个槽位栏，瓦片随机散回版面 | 第 5 关 |
| 炸弹 | 💣 | 选择一种图案，消除版面和槽位中所有该图案的瓦片 | 第 10 关 |
| 冰冻 | ❄️ | 冻结一块瓦片，接下来 3 次下落中不会被新瓦片覆盖 | 第 51 关 |

每通一关可以选择 **1 个道具**作为奖励。道具也可以不用选，直接进入下一关。

### 星级评价

每关根据表现获得 1–3 星：

| 星级 | 条件 |
|------|------|
| ★ | 完成关卡 |
| ★★ | 完成且步数不超过最优解的 1.5 倍 |
| ★★★ | 完成且步数不超过最优解的 1.2 倍，且未使用道具（提示除外） |

### 连击系统

连续消除相同图案的瓦片会积累连击。连击蓄力槽满 10 格时自动触发**闪电**——系统帮你消除一组最稀有的瓦片。连击只会增长，不会因为操作变慢或切换图案而清零。

### 高级关卡（第 51 关起）

从第 51 关开始，消除瓦片后会有新的瓦片从顶部掉落，增加节奏压力。掉落由你的操作驱动——消除得越多，掉落越频繁。

## 打开即玩

在浏览器中打开 `index.html` 即可开始游戏。推荐使用 Chrome、Safari 或 Edge。

也可以部署到任意静态托管服务（如 GitHub Pages），通过链接直接访问。

### 安装到手机主屏幕

在手机浏览器中打开游戏后，可以选择"添加到主屏幕"。安装后的体验接近原生 App：
- 独立窗口运行（没有浏览器地址栏）
- 支持完全离线游玩
- 有自己的图标和启动画面

## 游戏主题

游戏包含 6 套视觉主题，每 3 关自动轮换：

| 主题 | 图案风格 |
|------|----------|
| 缤纷果园 | 水果与农产品 |
| 奇趣动物园 | 各种动物 |
| 春日庭院 | 花草与自然 |
| 星际探险 | 太空与科技 |
| 环球美食 | 世界各地美食 |
| 像素冒险 | 游戏与冒险元素 |

每个主题有独立的图案库、背景图和背景音乐。

## 排行榜

游戏内置在线排行榜，不需要注册。首次打开时自动生成一个匿名身份，通关后成绩自动提交。

排行榜包含两个维度：
- **全球榜** — 按所有关卡的总星数排名
- **本关榜** — 按单关星级和步数排名

排行榜需要联网。离线时通关成绩会保存在本地，下次联网时自动提交。

## 设置

游戏内提供以下设置选项：
- **音效开关** — 独立控制游戏音效
- **背景音乐开关** — 独立控制 BGM
- **显示名称** — 排行榜中显示的名字，可自定义为 1–12 个字符

游戏过程中可随时点击右上角暂停按钮进入设置。

## 技术信息

- 渲染引擎：PixiJS v7.3.2（WebGL / Canvas）
- 音效：Web Audio API 程序化合成
- 离线支持：Service Worker
- 后端：Supabase 免费层
- 部署：纯静态文件，可托管在任意 Web 服务器
- 构建：零构建，无需 npm install 或打包工具
