# PWA、触觉反馈与动画优化

## PWA 支持

- 添加 `manifest.json` 和 Service Worker
- 可安装到手机主屏幕，类似原生应用的体验
- 支持离线游玩（排行榜除外）
- 自定义启动画面和应用图标

## 触觉反馈（振动 API）

- 瓦片点击：轻微振动（10ms）
- 三消：中等振动（30ms）
- 关卡完成：有节奏的振动模式
- 关卡失败：长振动（50ms）

注意：振动 API 仅在部分浏览器支持（如 Android Chrome）。实现优雅降级。

## 流畅动画

使用 PixiJS 渲染引擎，通过以下方式优化动画体验：
- 所有瓦片移动基于 PIXI.Ticker（requestAnimationFrame 封装），GPU 加速渲染
- 瓦片到槽位的轨迹使用贝塞尔曲线插值，模拟物理感
- 消除粒子效果使用自定义粒子池，预分配避免 GC 卡顿
- 连击屏幕闪烁使用 PIXI.Graphics overlay + alpha 动画
- 版面区域支持缩放，移动端可通过缩放按钮适配大关卡
