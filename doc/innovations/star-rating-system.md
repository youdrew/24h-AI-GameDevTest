# 星级评分系统

引入 **1–3 星**评分以增加重玩价值：

| 星星  | 条件                                                    |
|-------|---------------------------------------------------------|
| ⭐    | 完成关卡                                                |
| ⭐⭐   | 完成 + 步数 ≤ 最优步数 × 1.5                            |
| ⭐⭐⭐  | 完成 + 步数 ≤ 最优步数 × 1.2 + 未使用道具（提示除外）   |

- **提示不计为"使用道具"**：鼓励玩家在卡关时使用提示，不影响 3 星评价
- **洗牌/撤销/炸弹/冰冻**使用则无法获得 3 星
- 最优步数由求解器在关卡生成时计算（见 `discussion/difficulty-formula.md`）
- 星级在关卡完成界面显示。已获得最高星级保存到存储中
