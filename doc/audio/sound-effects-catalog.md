# 音效目录

## 场景

| 场景           | 描述                                           | 优先级 | 来源方式           |
|----------------|------------------------------------------------|--------|--------------------|
| 背景音乐       | 轻松循环 BGM，15-30 秒无缝循环                 | P0     | 音频文件（bgm.ogg + bgm.m4a）|
| 瓦片点击       | 清脆短促的点击声（< 200ms）                    | P0     | 程序化（Web Audio）|
| 瓦片飞入槽位   | 轻盈的移动声，有上升感                         | P0     | 程序化             |
| 三消           | 满足感的消除音效，像和弦                        | P0     | 程序化             |
| 关卡完成       | 欢快的胜利小曲（2–3 秒）                       | P0     | 音频文件           |
| 关卡失败       | 柔和的失败音，不刺耳                           | P0     | 程序化             |
| 洗牌           | 快速的洗牌声，密集                             | P1     | 程序化             |
| 撤销           | 反向/弹回音效，有倒退感                         | P1     | 程序化             |
| 提示           | 柔和的提示音，轻柔的闪光                       | P1     | 程序化             |
| 槽位警告       | 槽位栏接近满时的紧张提示                       | P1     | 程序化             |
| 瓦片下落       | 重力感的下落音                                 | P1     | 程序化             |
| 道具获取       | 获得道具时的清脆提示音                         | P1     | 程序化             |
| 层显           | 上层瓦片移除后下层变亮                         | P2     | 程序化             |
| 扔垃圾         | 清空槽位时的洗牌音效                           | P1     | 程序化（复用洗牌音效） |

## 连击系统音效

| 连击数 | 视觉文本 | 音效描述                              |
|--------|----------|---------------------------------------|
| x2     | Great!   | 基础消除音 + 音调升高半音             |
| x3     | Excellent! | 基础消除音 + 音调升高 2 半音        |
| x4     | Unbelievable! | 基础消除音 + 音调升高 3 半音      |
| x5+    | Combo Master! | 特殊和旋 + 闪电音效（自动消除 1 组同色） |

> 语音（SpeechSynthesis）已在 AI 自动决策中移除，连击文本以视觉动画呈现，无语音文件。

**每次消除操作均有音效反馈**，包括：
- 点击瓦片时的点击音
- 瓦片飞入槽位的移动音
- 消除成功时的三消音
- 连击时的递进音调
