# 核心规则

## 基本机制

1. **版面**：瓦片堆叠显示在屏幕上方，每块瓦片上带有图案
2. **操作**：玩家点击可用的瓦片，瓦片飞到底部"槽位栏"（收集槽）
3. **消除**：当槽位栏中出现 **3 块相同图案的瓦片**时，自动消除
4. **胜利**：清空版面所有瓦片
5. **失败**：槽位栏填满（7 格）且无法匹配

## 瓦片可用性

- 版面上的瓦片以**多层**堆叠
- 只有**未被其他瓦片覆盖**的瓦片才能点击
- 被覆盖的瓦片显示为半透明 / 不可交互

**覆盖逻辑**：当另一块瓦片在 X/Y 平面与其重叠且 Z 值更高时，该瓦片被视为"被覆盖"。每次消除后必须实时重新计算覆盖关系。

## 关卡进阶

- 多个难度递增的关卡
- 难度通过以下方式提升：
  - 更多瓦片
  - 更多堆叠层
  - 更多图案类型
  - 更紧凑的可用地空间
