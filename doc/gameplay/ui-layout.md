# UI 布局

## 屏幕布局

```
+---------------------------+
|  Level 12    Stars: ***   |  <- 顶部状态栏（PixiJS）
+---------------------------+
|                            |
|     [tile] [tile]          |
|   [tile][tile][tile]       |  <- 版面区域（PixiJS Canvas）
|  [tile][tile][tile][tile]  |     （堆叠瓦片，GPU 渲染）
|   [tile][tile][tile]       |
|     [tile] [tile]          |
|                            |
+---------------------------+
|  [1🔄] [2↩️] [3💡]        |  <- 道具按钮（PixiJS）：1=shuffleBtn 2=undoBtn 3=hintBtn
+---------------------------+
| [slot][slot][slot][slot]   |  <- 槽位栏（PixiJS，最多7格）
| [slot][slot][slot]         |
+---------------------------+
```

## 渲染分层

| 层级         | 技术   | 内容                                   |
|--------------|--------|----------------------------------------|
| 游戏画面     | PixiJS | 版面、瓦片、槽位栏、道具按钮、状态栏、粒子特效 |
| UI 覆盖层    | DOM    | 排行榜面板、设置面板、菜单弹窗          |

所有游戏交互元素统一由 PixiJS 渲染。DOM 仅用于需要表单输入、文字选择等场景的覆盖层面板。

DOM UI 使用 `position: absolute` 叠加在 PixiJS Canvas 上方。

## 移动端适配

- 竖屏优先、单手操作
- 版面居中，瓦片大小 48×48px（超过 44px 最小触摸目标）
- 槽位栏固定在底部，永不遮挡
- 版面区域提供缩放按钮，适配高难度关卡的大版面
