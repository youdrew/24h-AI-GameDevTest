# 性能目标

| 指标           | 目标                        |
|----------------|-----------------------------|
| 首次加载       | < 3 秒（移动 4G）           |
| 动画帧率       | ≥ 60fps（WebGL 加速）       |
| 交互延迟       | < 100ms（点击到视觉反馈）   |
| 内存占用       | < 50MB                      |
| PixiJS 库加载  | ~80KB gzipped（CDN 缓存后 0） |
