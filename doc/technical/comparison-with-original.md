# 与原版 Tile Explorer 对比

| 功能        | 原版 Tile Explorer              | 本 Web 版本                     |
|-------------|---------------------------------|---------------------------------|
| 广告        | 有（插屏广告 / 激励视频广告）   | **无广告**                      |
| 道具来源    | 看广告获取道具                  | 每关通过后选择 1 个道具          |
| 内购        | 有                             | 无                              |
| 平台        | Android / iOS 原生              | 网页浏览器（GitHub Pages）      |
| 渲染        | 原生渲染                       | PixiJS（WebGL / Canvas）        |
| 离线        | 支持                           | 支持（PWA + Service Worker）    |
| 安装        | 应用商店                       | 直接浏览器访问                  |
| 排行榜      | Game Center / Google Play Games | Supabase（免注册）              |
| 评分        | 无星级系统                     | 1-3 星评分 + 步数排名           |
| 难度系统    | 固定关卡                       | 参数化公式（数学一致性保证）    |
| 道具种类    | 基础道具                       | 6 种（含扔垃圾、炸弹、冰冻）    |
| 高级机制    | 无下落压力                     | 51+ 关卡有瓦片下落时间压力      |
| 主题系统    | 固定视觉                       | 6 个主题轮换（每 3 关），各有独立 emoji 库、背景图、BGM |
| 音效        | 音频文件                       | 程序化合成（音效）+ 音频文件（BGM/关卡音效） |
